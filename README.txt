# Transparent Video Web Page

This HTML file displays a transparent video using dual MP4s (color + alpha mask).

## How to use

1. Upload `transparent-video.html` to any static hosting provider (Netlify, Vercel, GitHub Pages, etc).
2. Copy the public URL.
3. In Framer, add an **Embed â†’ Website** component and paste the URL.
4. Set the size of the embed to match the video (374x500).

This method is compatible with Safari and other browsers.

Files used:
- Color video: https://ucarecdn.com/2f90b881-23d5-4fc0-ad0b-ba34e5a1568e/videocolor.mp4
- Alpha mask: https://ucarecdn.com/01172e48-57a9-4c2a-afda-dc615b0a632f/videoalpha.mp4
